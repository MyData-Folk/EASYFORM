Générateur de Fiches Clients

Instructions

  1. Chargez votre fichier Excel.
  2. Une fenêtre apparaît : associez les colonnes de votre fichier aux champs de la fiche.
  3. Cliquez sur "Générer" pour voir l'aperçu.
  4. Personnalisez le style, le logo, etc. via le bouton Configuration.
